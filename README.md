# Grading System

A simple Java-based grading system for managing student grades and role-based access (student/teacher).

## Getting Started

Clone the repository:

```bash
git clone https://github.com/toni-toni-choppa/grading-system.git
cd grading-system
```

Open the folder in your preferred IDE (VS Code, NetBeans, IntelliJ, etc.).

## How to Run

- Open `Main.java`
- Click **Run**

## Features

- Student grade calculation
- Teacher grade editing
- Role-based access (student vs teacher)
